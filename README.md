# WhiteboardBot

## Objective
- Draw images onto a whiteboard

## Requirements
- Cheap
- Easy to make
- Quick to setup
- Clean package

## Functionality
- Actuate whiteboard marker
- Turn image into instructions

## Subsystems
- Marker actuator
- Image processor
- Instrution uploader
